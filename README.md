# Simple LDO Converter

This repository contains the Altium Designer project files for a compact low-dropout (LDO) regulator PCB.

## Components
The components used in this project's design are the following:
* Yageo Group CC0805KKX7R6BB105 - Ceramic Capacitor, Ceramic, 10V, 10% +Tol, 10% -Tol, X7R, 15% TC, 1uF, Surface Mount, 0805 **(x2)**
* JST SM02B-GHS-TB(LF)(SN) - Conn Shrouded Header HDR 2 POS 1.25mm Solder RA Side Entry SMD T/R **(x2)**
* Microchip MIC5317-3.3YM5-TR - Fixed LDO Voltage Regulator, 2.5V to 6V, 155mV drop, 3.3V/150mA out, SOT-23-5 **(x1)**

## Reference
